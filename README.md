# 🥏 ProIsPro – Disc Inventory

A personal disc golf bag tracker built as a static website hosted on **GitHub Pages**.

## Features

- 📦 **Add / Edit / Remove** discs from your bag
- 🔍 **Search & filter** by type (Putter · Midrange · Fairway · Distance)
- 🗂 **Sort** by name, type, weight, or date added
- 💾 Data is saved in your browser's **LocalStorage** – no account needed
- 📤 **Export** your bag to a JSON file to share with friends
- 📥 **Import** a friend's bag JSON and merge it with yours
- 📱 Fully **responsive** – works great on your phone at the course

## Fields tracked per disc

| Field | Example |
|-------|---------|
| Name | Destroyer |
| Manufacturer | Innova |
| Type | Distance Driver |
| Plastic | Star |
| Weight | 175 g |
| Color | Red |
| Condition | Good |
| Flight numbers | 12 / 5 / -1 / 3 |
| Notes | Free-text notes |

## Deployment (GitHub Pages)

Push the repo to GitHub, then go to **Settings → Pages** and set the source to your main branch (`/root`). The site will be live at `https://<your-username>.github.io/proispro/`.
